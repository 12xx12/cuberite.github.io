MCServer - A Custom Minecraft Server

What is MCServer? MCServer is a custom Minecraft server, written completely in C++. It's designed to be a fast, lightweight and easy to use replacement for the original server.

Download Current Dev Build

Below you can download the current version for your operating system. MCServer supports Windows, Linux, and MacOSX. If your platform is not in the list below, then there is probably a bug, and is being worked on.

  • Linux 32-bit Requirements: Linux 32-bit
  • Linux 64-bit Requirements: Linux 64-bit
  • Windows Requirements: Windows XP SP3, Windows Server 2003, Windows Vista, Windows 7 or Windows 8
  • Mac Requirements: Intel Mac OS X 64-bit
  • Raspberry Pi Hard Float Requirements: Raspbian HF (default OS)
  • Raspberry Pi Soft Float Requirements: Raspbian Soft Float or Debian Soft Float

Source Code

MCServer is an Open-Source project, and thus you can always download the latest version from GitHub and compile the binaries yourself.

You'll need cmake and a c++ compilier, please see COMPILING.md for details
