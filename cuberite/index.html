Cuberite is a custom Minecraft server, written completely in C++. It's designed to be a fast, lightweight, awesome and easy to use replacement for Mojang's server or Bukkit, and has an extensive Plugin API that allows for plugin development in the Lua programming language.

You can download a pre-compiled ready to run copy of Cuberite from the Builds page, and Cuberite supports Windows, Linux and OSX, and would also support Solaris and FreeBSD with minor changes to the source. If you're interested in adding compatibility for your favourite OS/system, hop on over to the GitHub and get coding!

If you're new here, and want to get started using the server right away, head over to the Getting Started guide for more information and compiled build downloads. For more advanced operation, check out the User's Manual.

Cuberite is an open source project, so anybody can contribute and make it better, through source code contributions, documentation, web design or monetary donations. If you're interested in helping out with the project, you can fork us on GitHub, and chat with the developers on the forum. You can also contribute by developing plugins, or give us a small donation.
